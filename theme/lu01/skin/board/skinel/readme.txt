스키넬 (SkinL)
==================

Platform: G5
Skin Type: board
Skin Name: 스키넬 
Skin URI: http://lovelyus.net/skins/g5/board/skinel
Version: 1.2.5
Description: 러블리어스 스킨, SkinL 입니다. 스키넬 테마 플러그인을 같이 사용하면 겔러리,웹진,방명록 등의 기능을 사용할 수 있고, 스키넬 스킨과 연동되는 다양한 플러그인이 개발되고 있습니다.
Author: Chongmyung Park

Skin License: GPLv2
Required G5 Version: 5.0b01
Tested up to: 5.0b03
Donation URI: http://lovelyus.net/donation
Tags: 스키넬, SkinL, Skinel

러블리어스에서 개발/사용중인 스킨입니다. 현재 러블리어스의 모든 게시판은 스키넬로 되어 있습니다.

모바일 스킨
----------
  - G5 의 mobile/skin/board 에 업로드 하시고 모바일 스킨으로 skinel 을 선택하시면 모바일에서도 사용 가능합니다.
